# Dominoyar

Dominoyar is a Persian, RTL-friendly domino scorekeeper for web, PWA, and Android.

This public repository is the product home for Dominoyar. It contains public-facing information, support docs, release notes, and issue tracking. The application source code is not published here.

## Links

- Web app: https://dominoyar.ir
- Android APK: https://dominoyar.ir/downloads/dominoyar-android-v1.1.1.apk
- GitHub profile: https://github.com/moroshani

## What It Does

- Quick scorekeeping for 2-player, 3-player, and 4-player 2v2 domino games.
- Persian RTL mobile-first interface.
- Web/PWA access for browser and installable use.
- Native Android app distribution.
- Account, profile, settings, theme, sound, and haptics flows.
- Session continuation for unfinished games.

## Android Release

- Current release: `1.1.1`
- Package: `ir.dominoyar.app`
- Distribution: direct APK now, market distribution planned.

## Support

Use GitHub Issues in this repository for public bug reports, support requests, and feature suggestions.

Before opening an issue, include:

- Platform: web, PWA, or Android
- Device and operating system
- Steps to reproduce the problem
- Screenshot or screen recording if useful
